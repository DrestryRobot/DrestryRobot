光耦继电器的端口包括电源正负极、GPL输入信号、NC、公用电源接口和NO。其工作原理是通过STM32芯片控制GPL口电压,实现信号输入对电源开关的控制。跳线帽可改变触发方式,接h代表高电平触发,接I代表低电平触发。

▷00:00 引言

▷00:08 光耦继电器的端口

光耦继电器左边三个端口为电源正负极和GPL输入信号,右边三个端口为NC、公用电源接口和NO。

▷00:21 工作原理

光耦继电器通过信号输入控制右边电源闭合开关和闭合,可理解为黑匣子。

▷00:45 控制方式

STM32芯片控制GPL口电压,通过程序设定高电平或低电平,改变电路状态。

▷02:09 跳线帽的功能

跳线帽可改变触发方式,接h代表高电平触发,接l代表低电平触发。

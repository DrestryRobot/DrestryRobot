硬件 光电开关
===============
光电开关的工作原理,包括检测区域、指示灯、距离调节按钮、工作模式(L模式和D模式)以及NPN型和PNP型的区别。

▷00:01 引言

▷00:09 检测区域

光电开关的检测区域是椭圆形区域的上下两部,会发出平行的红外光线线源检测物体。

▷00:23 指示灯与距离调节按钮

指示灯指示物体存在与否,距离调节按钮可改变检测距离。

▷00:49 工作模式

工作模式有两种,L模式和D模式,分别实现高低电平的转变。

▷05:51 NPN型与PNP型

光电开关分为NPN型和PNP型,NPN型的曲线在负极和信号线之间,PNP型的曲线在正极和信号线之间。

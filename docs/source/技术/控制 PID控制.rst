控制 PID控制
===============
PID控制原理在无人机高度控制中的应用。通过比例参数P、积分参数I和微分参数D的调整,实现无人机从0米高度上升到100米高度的稳定控制。P根据偏差调整电流值,l对偏差积分使无人机接近目标高度,D对偏差微分产生阻尼效果,缩短稳定时间。

▷00:00 引言

▷00:13 案例描述

无人机从0米上升到100米,产生偏差e,PID out为无人机电机设定电流值。

▷00:53 比例参数P

P为比例参数,根据偏差调整电流值,使无人机接近目标高度。

▷02:38 积分参数I

I为积分参数,对偏差积分,使无人机超过目标高度后再下降,逐渐接近目标高度。

▷03:48 微分参数D

D为微分参数,对偏差微分,产生阻尼效果,缩短无人机稳定时间。

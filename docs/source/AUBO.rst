AUBO
------------
AUBO，是遨博智能开发的系列机器人。官方提供了相关的SDK开发包，可对Aubo机器人进行基于C++、C#、Python等多种语言的二次开发，充分释放机器人的生产力，提高生产作业的自动化和智能化水平。

欢迎来到 Lumache 的文档！
===================================

**Lumache** (/lu'make/) 是一个为厨师和美食爱好者设计的 Python 库，用于创建混合随机食材的食谱。  
它从 `Open Food Facts 数据库 <https://world.openfoodfacts.org/>`_ 中提取数据，并提供了一个*简单*且*直观*的 API。

请查看 :doc:`usage` 部分，了解更多信息，包括如何 :ref:`installation` 项目。

.. note::

   该项目正在积极开发中。

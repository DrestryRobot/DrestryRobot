SolidWorks 修改原点
====================
如何将模型的几何中心或其他点移动到世界原点的方法,包括新建装配体、导入模型、建立参考面、使用配合关系、保存为Pad格式文件等步骤。

▷00:00 引言

▷00:14 导入模型

关闭模型文件,新建装配体,导入模型并取消固定。

▷00:26 建立参考

在模型上建立参考面,如中心参考面。

▷00:50 配合关系

使用配合关系,使模型底面与世界原点重合,再使模型中心面与世界原点重合。

▷01:42 保存与转换

保存为Pad格式文件,原点位置改变,可移动到模型任意位置。

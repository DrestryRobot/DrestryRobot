Pixabay
---------
Pixabay，是一个国外免费的媒体素材资源下载网站，涵盖照片、插画、向量、视频、音乐、声音特效、GIF动画等多个类别的素材资源。还可注册并登陆自己的账号，添加素材到自己的媒体收藏库。

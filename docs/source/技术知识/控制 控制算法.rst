PID控制
---------
PID控制原理在无人机高度控制中的应用。通过比例参数P、积分参数I和微分参数D的调整,实现无人机从0米高度上升到100米高度的稳定控制。P根据偏差调整电流值,l对偏差积分使无人机接近目标高度,D对偏差微分产生阻尼效果,缩短稳定时间。

▷00:00 引言

▷00:13 案例描述

无人机从0米上升到100米,产生偏差e,PID out为无人机电机设定电流值。

▷00:53 比例参数P

P为比例参数,根据偏差调整电流值,使无人机接近目标高度。

▷02:38 积分参数I

I为积分参数,对偏差积分,使无人机超过目标高度后再下降,逐渐接近目标高度。

▷03:48 微分参数D

D为微分参数,对偏差微分,产生阻尼效果,缩短无人机稳定时间。

双环PID
----------
双环PID控制在无人机高度控制中的应用。单环PID控制可能导致加速度过大,而双环PID通过引入速度环,限制速度,使无人机上升过程更平稳,避免水洒出。

▷00:00 引言

▷00:10 单环PID

以无人机为例,单环PID通过位置偏差计算电机电流,实现高度控制,但加速度变化大,可能导致水洒出。

▷01:49 双环PID

双环PID引入速度环,限制速度,避免加速度过大,使无人机上升过程更平稳。

重力补偿
-----------
重力补偿算法包括负载辨识和重力补偿两部分。通过输入多组机械未端姿态数据和六维力传感器数据,进行最小二乘法拟合运算,得到重力补偿参数。再将实时数据输入重力补偿算法,减去重力分力,得到补偿值。补偿值越接近0,补偿效果越好。

▷00:01 引言

▷00:07 演示

点击负载辨识按钮,获取重力补偿参数,再输入机器人末端姿态数据和六维力传感器测量力矩数据,得到重力补偿结果。

▷00:30 重力补偿的原因

机器人未端负载重力在不同姿态下对六维力传感器产生的分力不断变化,通过减去重力分力,得到重力补偿后的值,越接近0,补偿效果越好。

01:06 算法实现

将多组机械末端姿态数据和六维力传感器数据输入负载辨识算法,得到重力补偿参数,再将实时数据输入重力补偿算法,得到补偿值。

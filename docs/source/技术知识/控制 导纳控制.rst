重力补偿
-----------
重力补偿算法包括负载辨识和重力补偿两部分。通过输入多组机械未端姿态数据和六维力传感器数据,进行最小二乘法拟合运算,得到重力补偿参数。再将实时数据输入重力补偿算法,减去重力分力,得到补偿值。补偿值越接近0,补偿效果越好。

▷00:01 引言

▷00:07 演示

点击负载辨识按钮,获取重力补偿参数,再输入机器人末端姿态数据和六维力传感器测量力矩数据,得到重力补偿结果。

▷00:30 重力补偿的原因

机器人未端负载重力在不同姿态下对六维力传感器产生的分力不断变化,通过减去重力分力,得到重力补偿后的值,越接近0,补偿效果越好。

01:06 算法实现

将多组机械末端姿态数据和六维力传感器数据输入负载辨识算法,得到重力补偿参数,再将实时数据输入重力补偿算法,得到补偿值。

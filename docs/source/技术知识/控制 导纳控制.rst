导纳控制
----------
导纳控制的概念、公式和核心参数,通过代码实现导纳控制的过程,以及对DKM参数的理解。实际应用中,导纳控制可实现用力拖机械臂,但效果可能受延迟等因素影响。

▷00:03 导纳控制的定义

手施加力,机械臂产生位移,未端越跟手,导纳控制越好。

▷00:26 公式及核心参数

导纳控制公式核心是力和位移,涉及刚性系数、阻尼系数和惯性系数。

▷01:19 代码实现

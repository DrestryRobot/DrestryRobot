设备通信入门
-------------

CAN通信
---------
控制电机的基本原理包括:控制器通过信号线发送电流值给电子调速器,电子调速器控制电机电流。为实现多电机控制,控制器需发送标志符给电子调速器,以区分不同电机。在PID控制中,电机需反馈角度、速度等信息给控制器,同样需发送标志符。

▷00:00 控制电机的基本原理

控制器通过信号线发送电流值给电子调速器,电子调速器控制电机电流。

▷00:34 电流值的转换与发送

控制器将电流值转换为二进制,通过高低电平变化发送给电子调速器。

▷02:48 多电机控制与标志符

通过标志符区分不同电机,实现多电机控制。

▷05:30 PID控制中的反馈

电机通过反馈标志符向控制器发送角度、速度等信息。

串口通信
------------
串口通信是一种协议,通过TXD和RXD线实现设备间的数据传输。与CAN通信不同,串口通信是单条路线,适用于简单应用。在实际应用中,可将不同协议理解为通用架构,根据需求进行选择和配置。

▷00:00 引言

串口通信是一种协议,理解协议对串口通信很重要。

▷00:46 TXD和RXD

TXD和RXD分别是发送和接收数据的线,设备间通过物理层面的线连接。

▷01:52 串口通信与其他通信的区别

串口通信是单条路线,与看通信不同,看通信是共线的。

▷02:30 串口通信的理解

串口通信协议在简单应用中差别不明显,但在复杂通信功能中需注意区别。

▷03:29 应用场景

以机械臂和力传感器通信为例,说明串口通信应用场景,可将不同协议理解为通用架构。

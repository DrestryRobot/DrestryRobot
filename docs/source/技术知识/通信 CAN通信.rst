控制 CAN通信
===============
控制电机的基本原理包括:控制器通过信号线发送电流值给电子调速器,电子调速器控制电机电流。为实现多电机控制,控制器需发送标志符给电子调速器,以区分不同电机。在PID控制中,电机需反馈角度、速度等信息给控制器,同样需发送标志符。

▷00:00 控制电机的基本原理

控制器通过信号线发送电流值给电子调速器,电子调速器控制电机电流。

▷00:34 电流值的转换与发送

控制器将电流值转换为二进制,通过高低电平变化发送给电子调速器。

▷02:48 多电机控制与标志符

通过标志符区分不同电机,实现多电机控制。

▷05:30 PID控制中的反馈

电机通过反馈标志符向控制器发送角度、速度等信息。

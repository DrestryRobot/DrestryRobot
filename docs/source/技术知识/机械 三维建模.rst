机械 三维建模
===============

基本概念
-----------
三维建模需要根据机器人的功能需求设计相应的三维模型，采用自上而下的设计思路，先确定机器人整体设计方案，划分机器人各功能机构，再对各功能机构进行方案设计，依据详细的设计方案对各零部件进行三维建模，最后进行各机构的装配及机器人整体组装。

技术要点
-----------
对于机器人本体结构的三维建模，在设计之初，必须明确的是机器人的功能定位，这为之后的所有设计奠定了基础，比如，扫地机器人的功能定位在于扫地，那么在结构设计上我们就能清楚，其大体应该由可移动底盘和清洁机构两个部分组成，再比如，人形机器人是仿照人的行为动作来设计的，故其在结构上应该具有头、身体、双臂、双腿足等部分。由此，我们很明显可以看出来，对于机器人本地结构的三维建模，一旦我们明确了其功能定位，那么根据功能定位我们就能确定其的大体结构设计组成。

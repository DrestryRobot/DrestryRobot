AuboUnity
==========

利用Unity搭建的第一个控制机械臂的仿真软件AuboUnity，欢迎下载体验。

AuboUnity程序压缩包下载链接：
--------------------------------
`点击此处下载压缩包 <zip/AuboUnity.zip>`_

压缩包内容：
------------
- 文件1：My project.exe
- 文件2：UnityCrashHandler64.exe
- 文件3：UnityPlayer.dll
- 文件4：MonoBleedingEdge
- 文件5：My project_Data

AuboUnity程序压缩包下载解压后，打开文件夹，双击``My project.exe``运行即可。

光轴导轨
---------
光轴导轨，是一种移动副标准零部件，一般由光轴、支架和滑块三部分组成。相比于线性导轨，光轴导轨安装所取空间较大，尤其是固定底平面与固定上平面的高度差比较大，设计中要求预留足够空间。

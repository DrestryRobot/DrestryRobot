Qt
----
Qt，是一个软件界面设计软件，本人所在教研室工业自动化设备软件开发的主要工具，其不仅支持编写基于多种语言的工程项目代码，还可开发可视化交互界面，同时Qt自带超多的库，方便直接引用，比如Qstring、Qmath等。

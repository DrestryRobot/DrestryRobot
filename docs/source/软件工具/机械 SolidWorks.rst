SolidWorks入门
------------------
SolidWorks是一个机械设计软件，主要包括零件设计、装配设计和工程图绘制三个核心板块。

零件设计包括草图绘制、特征操作、曲面设计等等，常见的功能有，草图的点、线、矩形、圆、圆弧、多边形等绘制，特征的拉伸、旋转、镜像等操作。

装配设计包括导入、配合零件等功能，其中，配合功能利用几何、机械等基本的零件配合方式将所有零件组合为装配体。

工程图绘制包括创建视图、添加注释等核心功能，其中，创建视图包括了常见的三视图、投影视图、剖视图、局部视图等，添加注释包括了粗糙度、几何公差、注释文本、零件序号、材料明细表等。

SolidWorks软件的入门介绍,包括菜单栏、设置页面和导航栏。主要功能包括新建零件、装配体和工程图文件,以及如何在这些文件中进行草图绘制、拉伸处理、零件导入、几何关系配合约束、视图描述和注释添加等操作。

SolidWorks警告消除
--------------------
在装配体中遇到警告报错问题时,可尝试压缩最近添加的零件或重新配合,也可压缩局部阵列配合以下的局部阵列,再解压压缩掉的配合。此外,压缩并解压多余零件可能有助于修复软件过载问题。

▷00:01 引言

▷00:17 常规解决方法

通过压缩最近添加的零件或重新配合来解决警告问题。

▷00:41 新的解决方法

压缩局部阵列配合以下的局部阵列,再解压压缩掉的配合,可解决警告问题。

▷01:41 总结

遇到警告时,压缩多余零件再解压,可能修复软件过载问题。

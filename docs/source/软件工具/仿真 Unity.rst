仿真 Unity
==================

Unity入门
------------
Unity大家都知道，很多平时玩的游戏都是基于Unity开发的，比较有意思有特点的是用Unity开发的游戏默认在程序开始时会显示Unity的商标动画，以表示这是基于Unity开发的。

在这里我们并不使用Unity作为游戏开发工具，而是作为机器人开发的数字孪生环节来做，近些年，Unity加大了对游戏之外，如工业数字孪生的功能支持，由此，我们使用其做机器人应用场景的数字孪生开发。

Unity主要功能包括模型文件导入、场景编辑、C#脚本开发等，其中，模型文件导入支持多种格式，覆盖动画人物、物体、字体等各种各样，而场景编辑就是对数字仿真场景添加光源、地面、相机、物体，然后对场景中所有东西进行属性编辑，更高级一点的，我们使用其C#脚本开发功能，可对场景中的物体进行物理学数字仿真。

Unity是一款广泛应用于游戏开发和工业数据孪生的软件。通过Unity Hub下载编辑器版本,新建项目并加载资源。编辑器界面包括菜单栏、目录树、属性栏和编辑窗口。资产管理包括导入模型、字体、纹理和脚本等。右键菜单可创建物体、光源和UI等。运行项目后进入游戏模式,可设置分辨率和显示内容。

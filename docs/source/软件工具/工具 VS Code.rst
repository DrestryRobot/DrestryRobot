VS Code入门
-------------
VS Code是一个免费、开源、跨平台的万能代码编辑器，支持多种编程语言。

VS Code主要包括资源管理器、源代码管理、运行和调试以及扩展等功能界面，资源管理器是主要编辑代码项目的地方，而源代码管理和git相结合可作为代码项目的历史版本管理工具，运行和调试功能可用于对单个代码脚本文件进行调试和运行，扩展则包括了各种各样功能丰富的插件，具体根据使用需要去里面找，然后下载安装就好了，非常方便。
